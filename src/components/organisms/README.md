# Organisms

Organisms adalah komponen yang terdiri dari beberapa atoms atau komponen lainnya.

## Tujuan
Membentuk bagian UI yang lebih kompleks.

## Contoh

- Navbar
- Sidebar
- ProductCard
- FormLogin

## Contoh Struktur

organisms/
 ├ Navbar.jsx
 ├ Sidebar.jsx
 ├ LoginForm.jsx
 └ ProductCard.jsx

## Catatan
Organisms biasanya menggabungkan atoms dan logic UI yang lebih kompleks.
